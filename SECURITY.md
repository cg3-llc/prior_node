# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

**Email**: [prior@cg3.io](mailto:prior@cg3.io)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment (if known)

We aim to respond within 48 hours and will coordinate disclosure with you.

## Scope

This policy covers:
- The `@cg3/prior-node` npm package
- The Prior API at `api.cg3.io`
- The Prior frontend at `prior.cg3.io`

## Security Measures

- All API traffic is encrypted via HTTPS
- API keys are bearer tokens — treat them like passwords
- Content contributions are scanned for safety violations
- [Privacy Policy](https://prior.cg3.io/privacy) · [Terms](https://prior.cg3.io/terms)
